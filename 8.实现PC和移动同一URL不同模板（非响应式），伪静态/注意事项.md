8.实现PC和移动同一URL不同模板（非响应式），伪静态
<br/>
8.1首先动态化，把根目录下生成的index.html删除，并且不再使用更新主页HTML-生成静态，如果根目录的index.php不见了，去安装包复制一个
<br/>
执行以下MySQL语句：
<br/>
update dede_archives set ismake=-1
<br/>
//将织梦所有栏目设置为"使用动态页"
<br/>
update dede_arctype set isdefault=-1
<br/>
//将网站所有文档都设置为"仅动态"
<br/>
